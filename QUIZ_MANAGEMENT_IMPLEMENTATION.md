# 测验管理功能实现

## 功能概述
为教师添加测验管理功能，让教师可以创建、编辑、发布测验，学生才能看到并参加。

## 需要实现的内容

### 后端 API（需要添加）
1. **创建测验** - POST /api/quizzes (教师)
2. **获取教师的测验列表** - GET /api/quizzes/teacher (教师)
3. **获取单个测验详情** - GET /api/quizzes/:id (教师)
4. **更新测验** - PUT /api/quizzes/:id (教师)
5. **删除测验** - DELETE /api/quizzes/:id (教师)
6. **发布/取消发布测验** - PATCH /api/quizzes/:id/toggle-active (教师)

### 前端页面和组件（需要创建）
1. **测验管理页面** - `/teacher/quizzes`
   - 测验列表
   - 创建测验按钮
   - 编辑/删除/发布操作

2. **测验表单组件**
   - 基本信息（标题、描述、时间限制）
   - 题目选择（多选）
   - 保存/取消按钮

3. **测验列表组件**
   - 显示测验卡片
   - 状态标识（已发布/草稿）
   - 操作按钮

### 路由更新
- 添加教师端测验管理路由
- 更新导航菜单
