## 🚀 快速使用

### 第一次使用
```bash
1. 双击 start-services.bat
2. 等待依赖安装和服务启动
3. 浏览器自动打开 http://localhost:3000
```

### 日常使用
```bash
启动：start-services.bat
停止：stop-services.bat
重启：restart-services.bat
检查：check-status.bat
```

## 💡 使用建议

### 开发流程
```
1. 启动服务
   → start-services.bat

2. 开发和测试
   → 在浏览器中测试功能
   → 查看命令行窗口的日志

3. 修改配置后重启
   → restart-services.bat

4. 结束工作
   → stop-services.bat
```

### 故障排除
```
1. 检查状态
   → check-status.bat

2. 如果服务异常
   → restart-services.bat

3. 如果端口被占用
   → stop-services.bat
   → 等待几秒
   → start-services.bat
```

## 📝 维护说明

### 修改端口
如果需要修改默认端口，需要同时更新：
1. `backend/.env` - 后端端口
2. `frontend/vite.config.ts` - 前端端口
3. 批处理脚本中的端口号

### 添加新服务
如果添加了新的服务（如 Redis），需要：
1. 在 `start-services.bat` 中添加启动逻辑
2. 在 `stop-services.bat` 中添加停止逻辑
3. 在 `check-status.bat` 中添加状态检查
