# 删除确认提示修改

## 修改内容

更新了删除测验时的确认提示文字，准确反映实际的删除行为。

## 修改的文件

### frontend/src/pages/teacher/QuizManagement.tsx

**修改前**：
```tsx
message="确定要删除这个测验吗？此操作无法撤销，已提交的答题记录将保留。"
```
❌ 错误：实际上会删除提交记录，但提示说会保留

**修改后**：
```tsx
message="确定要删除这个测验吗？此操作无法撤销，将同时删除所有相关的提交记录和分析数据。"
```
✅ 正确：准确描述了实际的删除行为

## 删除确认对话框

### 显示内容
- **标题**：确认删除
- **消息**：确定要删除这个测验吗？此操作无法撤销，将同时删除所有相关的提交记录和分析数据。
- **确认按钮**：删除
- **取消按钮**：取消

### 实际删除的数据
1. ✅ 测验本身
2. ✅ 所有学生的提交记录
3. ✅ 所有题目的 AI 分析数据

## 用户体验

### 修改前
用户可能误以为删除测验后，学生的答题记录会保留，导致误操作。

### 修改后
用户清楚地知道删除测验会同时删除所有相关数据，可以做出更明智的决定。
